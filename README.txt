DECATHLON E-COMMERCE WEBSITE
============================

A complete e-commerce website built with PHP, MySQL, and Bootstrap 5.
Features include user authentication, shopping cart, checkout, and admin panel.

SETUP INSTRUCTIONS
==================

1. Database Setup:
   - Open phpMyAdmin (http://localhost/phpmyadmin)
   - Create a new database named "decathlon_db"
   - Import the database_setup.sql file to create tables and sample data
   - Default admin credentials: admin@decathlon.com / admin123

2. File Structure:
   - /admin/ - Admin panel files
   - /assets/ - CSS, JS, and images
   - /config/ - Database configuration
   - /includes/ - Header, footer, and functions
   - /pages/ - Frontend pages

3. Access URLs:
   - Frontend: http://localhost/decathlon/
   - Admin Panel: http://localhost/decathlon/admin/

4. Features:
   - User registration and login
   - Product browsing and search
   - Shopping cart management
   - Checkout process
   - Order management
   - Admin dashboard
   - Product and category management
   - User management

5. Sample Data:
   - 5 categories
   - 20+ products
   - Admin user account

TECHNOLOGY STACK
================

- Backend: PHP 8+
- Database: MySQL
- Frontend: HTML5, CSS3, JavaScript
- Framework: Bootstrap 5
- Icons: Bootstrap Icons

SUPPORT
=======

For any issues, check:
1. Database connection settings in config/database.php
2. File permissions
3. XAMPP/Apache configuration
